# 论文的写法

摘要用来提出问题

相关工作指出缺点

方法指出我们提出的方法是怎么做的

实验我所加入的东西是有用的 对比实验和以前的模型比一比

总结是我的方法有多好

# 如何找文献？

- 有代码
- 代码跑通
- 代码效果能达到论文的效果，（用来对比（水平不到，能能跑就可以
